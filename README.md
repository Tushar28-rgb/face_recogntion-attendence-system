# 🎯 Face Recognition Attendance System

## 📌 Description
This project is an AI-based Face Recognition Attendance System that automatically detects and recognizes faces using a webcam and marks attendance in real-time. It eliminates manual attendance and improves accuracy and efficiency.

---

## 🚀 Features
- 🎥 Real-time face detection using webcam  
- 🧠 Face recognition using trained data  
- 📝 Automatic attendance marking in CSV file  
- ⚡ Fast and accurate system  
- 📂 Stores known faces for identification  

---

## 🛠️ Technologies Used
- Python  
- OpenCV  
- face_recognition library  
- NumPy
